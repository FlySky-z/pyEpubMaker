# pyEpubMaker

通过使用python，快速构建epub电子书。

## 介绍

> 通过使用python，你可以在短短的三行代码构建一个简易的epub电子书，让电子书的制作不再复杂。

*警告⚠️*：目前的代码处于搭建阶段，可能在使用过程中出现某些神奇问题。



## 详细

* epub电子书制作标准使用`v2.0`。
* 目前电子书尚未完成封装，制作出来的电子书需要手动打包成`.zip`后修改后缀为`.epub`。
